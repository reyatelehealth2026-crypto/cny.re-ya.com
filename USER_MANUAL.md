## 1. เริ่มต้นใช้งาน

### 1.1 ความต้องการของระบบ

| รายการ | ความต้องการ |
|--------|-------------|
| PHP | >= 7.4 |
| MySQL | >= 5.7 หรือ MariaDB >= 10.2 |
| Web Server | Apache / Nginx |
| SSL | จำเป็น (HTTPS) |
| PHP Extensions | PDO, cURL, JSON, Mbstring |

### 1.2 การติดตั้ง

**ขั้นตอนที่ 1: อัพโหลดไฟล์**
```bash
# อัพโหลดไฟล์ทั้งหมดไปยัง public_html หรือ www
```

**ขั้นตอนที่ 2: เปิด Installation Wizard**
```
https://yourdomain.com/install/
```

**ขั้นตอนที่ 3: ทำตามขั้นตอน**
1. ตรวจสอบ Requirements
2. กรอกข้อมูลฐานข้อมูล
3. ตั้งค่าระบบ
4. สร้างบัญชี Admin
5. ติดตั้งเสร็จสิ้น

### 1.3 การตั้งค่า LINE Messaging API

1. ไปที่ [LINE Developers Console](https://developers.line.biz/console/)
2. สร้าง Provider > สร้าง Channel (Messaging API)
3. คัดลอก **Channel ID**, **Channel Secret**, **Channel Access Token**
4. ตั้งค่า Webhook URL:
   ```
   https://yourdomain.com/webhook.php?account=1
   ```
5. เปิด **Use webhook** = ON
6. ปิด **Auto-reply messages** = OFF
7. ปิด **Greeting messages** = OFF

### 1.4 การตั้งค่า LIFF

1. ไปที่ LINE Developers Console > LIFF
2. สร้าง LIFF App ใหม่
3. ตั้งค่า Endpoint URL: `https://yourdomain.com/liff-main.php`
4. Scope: `profile`, `openid`
5. คัดลอก LIFF ID ไปใส่ในระบบ

## 2. สำหรับผู้ใช้ (ลูกค้า)

### 2.1 การเริ่มต้นใช้งาน

#### Add Friend LINE OA
1. สแกน QR Code หรือค้นหา LINE ID ของร้าน
2. กด "เพิ่มเพื่อน"
3. ได้รับข้อความต้อนรับอัตโนมัติ

#### สมัครสมาชิก
1. กดปุ่ม "สมัครสมาชิก" ใน Rich Menu หรือข้อความต้อนรับ
2. กรอกข้อมูล:
   - ชื่อ-นามสกุล (จำเป็น)
   - วันเกิด (จำเป็น)
   - เพศ (จำเป็น)
   - เบอร์โทร, อีเมล (ไม่บังคับ)
   - โรคประจำตัว, ยาที่แพ้ (ไม่บังคับ)
   - ที่อยู่ (ไม่บังคับ)
3. กด "ยืนยันสมัครสมาชิก"
4. ได้รับบัตรสมาชิกดิจิทัล + แต้มต้อนรับ

### 2.2 การสั่งซื้อสินค้า

#### ขั้นตอนการสั่งซื้อ
```
1. เปิดร้านค้า    →  กดปุ่ม "Shop" หรือ "ร้านค้า"
2. เลือกสินค้า    →  เลือกหมวดหมู่ > กดสินค้าที่ต้องการ
3. เพิ่มตะกร้า    →  เลือกจำนวน > กด "เพิ่มลงตะกร้า"
4. ตรวจสอบ       →  กดไอคอนตะกร้า > ตรวจสอบรายการ
5. Checkout      →  กรอกที่อยู่จัดส่ง > เลือกวิธีชำระเงิน
6. ชำระเงิน      →  โอนเงิน > อัพโหลดสลิป
7. รอยืนยัน      →  รอ Admin ตรวจสอบและจัดส่ง
```

#### วิธีชำระเงิน
| วิธี | รายละเอียด |
|------|------------|
| โอนเงิน | โอนเงินตามบัญชีที่แจ้ง แล้วอัพโหลดสลิป |
| PromptPay | สแกน QR Code ชำระเงิน |
| เก็บเงินปลายทาง | ชำระเงินเมื่อได้รับสินค้า |

#### การติดตามสถานะออเดอร์
- กดปุ่ม "ออเดอร์ของฉัน" หรือ "My Orders"
- ดูสถานะ: รอชำระ → ชำระแล้ว → กำลังจัดส่ง → จัดส่งแล้ว

### 2.3 การปรึกษา AI เภสัชกร

#### วิธีใช้งาน
```
พิมพ์: @ai ตามด้วยคำถาม
ตัวอย่าง: @ai ปวดหัวมียาอะไรบ้าง
         @ai ยาแก้ไอสำหรับเด็ก
         @ai วิตามินบำรุงสายตา
```

#### ฟีเจอร์ AI เภสัชกร
- ✅ ตอบคำถามเรื่องยาและสุขภาพ
- ✅ ค้นหาสินค้าอัตโนมัติ
- ✅ แสดงรูปสินค้าพร้อมราคา
- ✅ ตรวจสอบประวัติแพ้ยา
- ✅ ตรวจจับอาการฉุกเฉิน (Red Flag)

#### ข้อควรระวัง
- AI ให้คำแนะนำเบื้องต้นเท่านั้น
- หากมีอาการรุนแรง ควรพบแพทย์
- ข้อมูลยาที่แพ้จะถูกตรวจสอบอัตโนมัติ

### 2.4 การนัดหมาย

#### ขั้นตอนการจองนัดหมาย
1. กดปุ่ม "นัดหมาย" หรือ "Appointment"
2. เลือกประเภทบริการ (ปรึกษาเภสัชกร, Video Call, ฯลฯ)
3. เลือกวันที่และเวลา
4. กรอกอาการ/หมายเหตุ
5. ยืนยันการจอง

#### การแจ้งเตือน
- แจ้งเตือนก่อนนัด 1 วัน
- แจ้งเตือนก่อนนัด 1 ชั่วโมง

### 2.5 บัตรสมาชิกและแต้มสะสม

#### การดูบัตรสมาชิก
- กดปุ่ม "บัตรสมาชิก" หรือ "Member Card"
- แสดง QR Code สำหรับสะสมแต้ม
- แสดงแต้มคงเหลือ

#### การสะสมแต้ม
- ได้รับแต้มเมื่อซื้อสินค้า (ตามที่ร้านกำหนด)
- แต้มต้อนรับเมื่อสมัครสมาชิก

#### การแลกแต้ม
1. กดปุ่ม "แลกแต้ม" หรือ "Redeem"
2. เลือกของรางวัลที่ต้องการ
3. ยืนยันการแลก
4. แสดง QR Code ให้พนักงานสแกน

## 3. สำหรับ Admin

### 3.1 การเข้าสู่ระบบ

```
URL: https://yourdomain.com/auth/login.php
```

#### ระดับผู้ใช้
| Role | สิทธิ์ |
|------|--------|
| Super Admin | จัดการทุกอย่าง, ทุกบัญชี LINE |
| Admin | จัดการบัญชี LINE ที่กำหนด |
| User | จัดการบัญชี LINE ของตัวเอง |

### 3.2 Dashboard

เมื่อเข้าสู่ระบบจะเห็น Dashboard แสดง:
- 📊 สถิติผู้ใช้ทั้งหมด / ใหม่วันนี้
- 💬 ข้อความวันนี้
- 🛒 ออเดอร์ทั้งหมด / รอดำเนินการ
- 💰 รายได้รวม
- 📦 สินค้า / สินค้าใกล้หมด
- 🤖 Auto Reply ที่เปิดใช้งาน

### 3.3 การจัดการข้อความ (Messages)

#### เข้าถึง: เมนู "แชท" หรือ "Messages"

**ฟีเจอร์:**
- ดูรายชื่อลูกค้าที่ส่งข้อความ
- Badge แสดงข้อความที่ยังไม่อ่าน
- ดูประวัติการสนทนา
- ตอบกลับลูกค้าได้ทันที
- ดูรูปภาพ/ไฟล์ที่ลูกค้าส่งมา

**วิธีตอบกลับ:**
1. คลิกเลือกลูกค้าจากรายชื่อ
2. พิมพ์ข้อความในช่องด้านล่าง
3. กด Enter หรือปุ่มส่ง

### 3.4 การจัดการออเดอร์ (Orders)

#### เข้าถึง: เมนู "ร้านค้า" > "ออเดอร์"

**สถานะออเดอร์:**
| สถานะ | ความหมาย | Action |
|--------|----------|--------|
| 🟡 Pending | รอชำระเงิน | รอลูกค้าชำระ |
| 🟠 รอตรวจสลิป | ลูกค้าอัพโหลดสลิปแล้ว | ตรวจสอบและอนุมัติ |
| 🟢 Paid | ชำระเงินแล้ว | เตรียมจัดส่ง |
| 🔵 Shipping | กำลังจัดส่ง | รอลูกค้ารับสินค้า |
| ✅ Delivered | จัดส่งแล้ว | เสร็จสิ้น |
| ❌ Cancelled | ยกเลิก | - |

**การอนุมัติสลิป:**
1. คลิกที่ออเดอร์ที่มีสถานะ "รอตรวจสลิป"
2. ดูรูปสลิปที่ลูกค้าอัพโหลด
3. ตรวจสอบยอดเงินและวันที่
4. กด "อนุมัติ" หรือ "ปฏิเสธ"

**การอัพเดทสถานะ:**
1. คลิกที่ออเดอร์
2. เลือกสถานะใหม่
3. กรอกเลข Tracking (ถ้ามี)
4. กด "บันทึก"
5. ระบบจะแจ้งลูกค้าอัตโนมัติ

### 3.5 การจัดการสินค้า (Products)

#### เข้าถึง: เมนู "ร้านค้า" > "สินค้า"

**เพิ่มสินค้าใหม่:**
1. กดปุ่ม "+ เพิ่มสินค้า"
2. กรอกข้อมูล:
   - ชื่อสินค้า
   - รหัสสินค้า (SKU)
   - ราคา / ราคาโปรโมชั่น
   - หมวดหมู่
   - รูปภาพ
   - รายละเอียด
   - จำนวนในสต็อก
3. กด "บันทึก"

**นำเข้าสินค้าจาก CNY API:**
1. ไปที่ "ร้านค้า" > "นำเข้าสินค้า"
2. กด "Sync จาก CNY"
3. รอระบบดึงข้อมูล

### 3.6 การ Broadcast ข้อความ

#### เข้าถึง: เมนู "บรอดแคสต์"

**ประเภท Broadcast:**

|--------|------------|

**ขั้นตอนการส่ง Broadcast:**

2. สร้างเนื้อหา
3. เลือกกลุ่มเป้าหมาย:

   - เฉพาะ Tag

   - ส่งทันที
   - ตั้งเวลา

**Broadcast Catalog:**
1. ไปที่ "บรอดแคสต์" > "แคตตาล็อก"
2. กด "+ สร้างแคมเปญ"
3. เลือกสินค้าที่ต้องการโปรโมท (สูงสุด 10 รายการ)

### 3.7 การตั้งค่า Auto Reply

#### เข้าถึง: เมนู "แชทและลูกค้า" > "ตอบอัตโนมัติ"

**สร้างกฎใหม่:**
1. กด "+ เพิ่มกฎ"
2. กรอกข้อมูล:
   - Keyword: คำที่ต้องการให้ตอบ
   - Match Type:
     - `exact` = ตรงทั้งหมด
     - `contains` = มีคำนี้อยู่
     - `starts_with` = ขึ้นต้นด้วย
     - `regex` = Regular Expression
   - Response Type:
     - Text = ข้อความธรรมดา
     - Image = รูปภาพ
     - Flex = Flex Message
   - Response: เนื้อหาที่ต้องการตอบ
3. กด "บันทึก"

**ตัวอย่างกฎ:**
| Keyword | Match Type | Response |
|---------|------------|----------|
| สวัสดี | contains | สวัสดีค่ะ ยินดีให้บริการ |
| ราคา | contains | [Flex: รายการสินค้า] |
| โปรโมชั่น | exact | [Image: โปรโมชั่นเดือนนี้] |
| @ai | starts_with | → ส่งต่อ AI เภสัชกร |

### 3.11 การตั้งค่า Rich Menu

4. กด "บันทึก"
5. กด "ตั้งเป็น Default" เพื่อใช้งาน

## 4. การตั้งค่าระบบ

**เพิ่มบัญชี LINE ใหม่:**

3. กด "บันทึก"
4. ทดสอบการเชื่อมต่อ

### 4.2 ตั้งค่าร้านค้า

#### เข้าถึง: เมนู "ร้านค้า" > "ตั้งค่าร้าน"

- ชื่อร้าน
- โลโก้

**การชำระเงิน:**
- บัญชีธนาคาร

**การจัดส่ง:**

### 4.3 ตั้งค่า AI เภสัชกร

**ตั้งค่า:**
- เปิด/ปิด AI
- Gemini API Key
- System Prompt
- Temperature
- Max Tokens

### 4.4 ตั้งค่า Telegram แจ้งเตือน

#### เข้าถึง: เมนู "ตั้งค่า" > "Telegram"

**ขั้นตอน:**
1. สร้าง Bot ใน Telegram (@BotFather)

3. หา Chat ID ของกลุ่มหรือตัวเอง
4. กรอกข้อมูลในระบบ
5. ทดสอบการส่ง

**การแจ้งเตือน:**
- ลูกค้าใหม่ Add Friend
- ออเดอร์ใหม่
- อัพโหลดสลิป
- Red Flag จาก AI

### 4.5 ตั้งค่า Cron Jobs

**Cron Jobs ที่จำเป็น:**
```bash
# Sync สินค้า - ทุก 1 นาที
```

## 5. การแก้ไขปัญหา

### 5.1 Webhook ไม่ทำงาน

**ตรวจสอบ:**
1. URL เป็น HTTPS หรือไม่
2. Channel Secret ถูกต้องหรือไม่
3. Webhook URL ตรงกับที่ตั้งค่าใน LINE Developers

**วิธีแก้:**
```php
// ดู error log
tail -f /path/to/error_log
```

### 5.2 ส่งข้อความไม่ได้

**ตรวจสอบ:**
1. Channel Access Token ยังไม่หมดอายุ
2. Token มีสิทธิ์ส่งข้อความ
3. ผู้ใช้ยังไม่ Block บอท

### 5.3 LIFF ไม่ทำงาน

**ตรวจสอบ:**
1. LIFF ID ถูกต้อง
2. Endpoint URL ตรงกับที่ตั้งค่า
3. เปิดผ่าน LINE App (ไม่ใช่ Browser ภายนอก)

### 5.4 รูปภาพไม่แสดง

**ตรวจสอบ:**
1. URL เป็น HTTPS
2. ไฟล์มีอยู่จริง
3. Permission ของโฟลเดอร์ uploads

### 5.5 Sync สินค้าไม่ทำงาน

**ตรวจสอบ:**
1. Cron Job ทำงานหรือไม่
2. API Key ถูกต้อง
3. ดู Sync Dashboard สำหรับ error
